Arpentons — Email Design Reference

Ce fichier sert de référence pour la charte graphique des emails Arpentons. Il est utilisé pour générer les templates Brevo à chaque nouvel événement.

Typographie

Titres : Libre Caslon Text (Google Fonts) — fallback : Georgia, serif

Corps : Hanken Grotesk (Google Fonts) — fallback : Arial, sans-serif

Taille titre principal : 28px, weight 700

Titre section : 20px, weight 600

Corps : 15-16px, line-height 1.6

Label/tag : 11px, uppercase, letter-spacing 0.1em, weight 600

Structure email

  • Largeur max : 600px, centré
  • Header : fond primary (#182232), titre blanc Libre Caslon, tag "LA ROCHELLE" fond secondary (#9d4400)
  • Corps : fond blanc, padding 40px 30px
  • Blocs info (date/lieu) : fond surface (#f0eee9), border 1px solid #c5c6cd
  • Blocs livre : fond #fbf9f4, border-left 4px solid #ffdbca
  • Bloc alerte : fond secondary (#9d4400), texte blanc
  • Footer : fond primary (#182232), texte #fbf9f4, lien secondary (#ffdbca)
  • Pas d'ombres épaisses — style épuré
